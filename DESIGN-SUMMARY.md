# Design summary

- Design concept: Payroll compliance binder
- Layout personality: traditional institutional
- Typography: EB Garamond headings with Inter body text
- Color palette: primary #713f12, secondary #fde68a, accent #1d4ed8, background #fffbeb, text #1c1917, muted #78716c
- Shape language: Slight rounding
- Visual motif: binder tabs and payroll calendar
- Unique layout choices: first sections use Payroll binder, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
